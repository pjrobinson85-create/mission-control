## How to Use It

### Medication
- Morning: 9:00 AM reminder fires
- You take meds
- You type: `took meds` (optional, stops reminders escalating)

### When Stuck
- Type: `I'm stuck`
- I ask: What's unclear?
- We debug + pick next step

### Asking for Tasks
- `quick win` → Give me a 10–15 min task
- `high priority` → Show NDIS + Report Writer only
- `report writer` → Filter to report writer tasks
- `show my tasks` → Everything, by priority

### Working
- Optional: `starting work` (tells me you're in flow)
- I check in every 45 mins (soft suggestion)
- Type: `done, taking break` (resets timer)

### Other Stuff
- Bowel routine reminder fires Sun/Wed at 8:20 PM
- Night agents post findings each morning
- You read findings, decide next action
