# Quick Commands for Paul

Use these anywhere in the chat to interact with your assistant system.

## Medication & Health

- **`took meds`** → Tell me you've taken your meds (I'll log it and stop nagging)
- **`skip meds`** → Mark meds as skipped (I'll note the consequences for tomorrow)
- **`meds status`** → Check if meds were taken today

## Tasks & Work

- **`show my tasks`** → List all tasks by project
- **`high priority`** → Show critical tasks (NDIS, health)
- **`quick win`** → Give me something I can finish in <15 mins
- **`10 min task`** → What can I do in exactly 10 minutes?
- **`NDIS work`** → Show highest priority NDIS tasks
- **`add task`** → Create a new task (I'll ask for details)

## Focus & Breaks

- **`starting work`** → Tell me you're hyperfocusing, I'll check in periodically
- **`done, taking break`** → Log that you took a break (resets hyperfocus timer)
- **`remind me in X mins`** → Set a one-off reminder for something

## Feeling & Mood

- **`I'm stuck`** → Tell me you're procrastinating, let's troubleshoot
- **`I feel like crap`** → Log how you're feeling (energy, focus, pain, etc.)
- **`mood check`** → How are you doing today?

## Reports & Insights

- **`what did I do this week`** → Summary of completed tasks
- **`meds compliance`** → How often did I take meds this week?
- **`productivity report`** → Week/month breakdown

---

## How It Works

- **Ramping reminders:** Start gentle, escalate if ignored, mention consequences
- **Hyperfocus check:** Every 45 mins I'll ask if you need a break
- **Smart nudges:** Procrastinating? I'll call you out and offer easy wins
- **No judgment:** If you skip meds or miss tasks, I log it and we move on. The goal is to help, not shame.

This is flexible. Tell me what you need and we'll adjust.
