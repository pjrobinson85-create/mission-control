## The 5 Tools

### 1. **AgentMail** (Email for Agents)
**URL:** agentmail.to  
**What it does:** Send emails from your agent without Gmail bot detection  
**Setup:**
1. Go to agentmail.to
2. Click "Get Started"
3. Create account
4. Get API key from "API Keys" section
5. Give key to your OpenClaw agent
6. Agent can now send emails natively

**Why it's cool:** Gmail bans bot-detected senders; AgentMail is built specifically for agents.  
**Pricing:** Free

---

### 2. **Here.Now** (Instant Hosting)
**URL:** here.now  
**What it does:** Instant URL hosting for websites, documents, anything  
**Setup:**
1. Go to here.now
2. Tell agent: "check out here.now and get that installed"
3. OR: Copy setup instructions and paste into agent
4. Agent can now publish instantly

**Features:**
- 24-hour URLs (no account needed)
- Permanent URLs (with free account)
- Zero-backend hosting
- Perfect for sharing docs, sites, marketing plans

**Use case:** Agent creates a document → host on here.now → get shareable URL  
**Pricing:** Free

---

### 3. **Tavily** (Web Search)
**URL:** tavily.com  
**What it does:** Free web search API for agents (faster than Brave)  
**Setup:**
1. Go to tavily.com
2. Sign up for free account
3. Copy API key
4. Give to your agent

**Features:**
- ~1,000 free searches/month
- Faster than Brave browser
- Built for agents
- Clean results

**Why not Brave:** Brave now charges $3-5/month; Tavily stays free  
**Pricing:** Free (then pay-as-you-go)

---

### 4. **FireCrawl** (Web Scraping)
**URL:** firecrawl.dev  
**What it does:** Turn any website into clean, AI-ready data  
**Setup:**
1. Go to firecrawl.dev
2. Login/create free account
3. Copy API key
4. Tell agent: "Use FireCrawl whenever you scrape a website"

**Features:**
- Tons of free credits
- Extracts clean data (removes junk, ads, etc.)
- Track usage in dashboard
- Perfect for competitor research

**Use case:** 
- Get competitor pricing
- Extract product details
- Research website content
- Turn messy HTML → clean text

**Pricing:** Free (with generous credits)

---

### 5. **Remotion** (Programmatic Video Generation)
**URL:** remotion.dev  
**What it does:** Create videos programmatically (talk-to-video)  
**Setup:**
1. Tell agent: "Download Remotion"
2. Agent handles the rest

**Features:**
- Create videos by talking (no manual editing)
- Professional output
- Free forever (free tier sufficient)
- Edit by voice: "make my photo bigger", "change color", etc.

**Use cases:**
- Promo videos
- Milestone announcements
- Instagram story clips
- Short video snippets

**Example:** Agent made a "3,000 YouTube subscribers" promo video in seconds.  
**Pricing:** Free forever
