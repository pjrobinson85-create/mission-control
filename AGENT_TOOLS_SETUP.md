# Quick Setup: 5 Agent Tools

Copy each API key below and provide to your OpenClaw agent.

---

## 1️⃣ AgentMail (Email)

**Setup for Agent:**
```
I have an AgentMail API key. Can you integrate it so I can send emails?
API Key: [PASTE_YOUR_KEY]
```

**Get Your Key:**
1. agentmail.to → Sign up
2. Account → API Keys → Copy

---

## 2️⃣ Here.Now (Hosting)

**Setup for Agent:**
```
Install Here.Now so I can publish websites instantly.
Go to here.now and copy the setup instructions, then integrate them.
```

**No API key needed** — Agent will handle setup via website.

---

## 3️⃣ Tavily (Web Search)

**Setup for Agent:**
```
I have a Tavily API key for web search. Integrate it so you can search the web.
API Key: [PASTE_YOUR_KEY]
```

**Get Your Key:**
1. tavily.com → Sign up
2. Account → API Keys → Copy

---

## 4️⃣ FireCrawl (Web Scraping)

**Setup for Agent:**
```
I have a FireCrawl API key. Use it whenever I ask you to scrape a website.
API Key: [PASTE_YOUR_KEY]
```

**Get Your Key:**
1. firecrawl.dev → Sign up
2. Account → API Keys → Copy

---

## 5️⃣ Remotion (Video Generation)

**Setup for Agent:**
```
Download and integrate Remotion so I can create videos programmatically.
I want to be able to talk-to-video (voice to edit).
```

**No API key** — Remotion is installed locally.

## Pro Tip

Once integrated, you can tell your agent things like:

**Email:** "Send an email to paul@example.com saying..."  
**Hosting:** "Create a landing page and host it on Here.Now"  
**Search:** "Search the web for latest AI tools"  
**Scraping:** "Scrape competitor.com and extract their pricing"  
**Video:** "Make a 15-second promo video with this image and text"

Agent handles the rest. 💯

---

## Recommended Workflow

1. **Tavily** (find pages) → **FireCrawl** (extract data) → **Here.Now** (share results)
2. **AgentMail** (send findings) to stakeholders
3. **Remotion** (create summary video)

All in one agent prompt. ✨
