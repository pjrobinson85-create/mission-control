Steps to Access from Another Device
  • Open a web browser on another device on your network
  • Go to the LAN address shown above (or your custom IP)
  • You'll see Mission Control dashboard
  • Data automatically syncs across all connected devices

Find Your Server IP

Run this on your server machine:

Linux/Mac
ifconfig | grep "inet " | grep -v 127.0.0.1
Windows
ipconfig

Look for an IPv4 address like 192.168.1.xxx or 10.0.0.xxx

Troubleshooting

Can't connect from another device?

  • Make sure both devices are on the same network
  • Check firewall isn't blocking port 8899
  • Verify the IP address matches your server
  • Try pinging the server: ping 192.168.1.100

Server not showing LAN IP on startup?

  • Check server is running: node server.js
  • Should show both "Local" and "LAN" addresses
  • Use the LAN address from the startup message
